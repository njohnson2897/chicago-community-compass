# Chicago Community Compass

A comprehensive mapping platform connecting Chicago residents with essential social services. Find food banks, shelters, healthcare services, and other community resources in your area.

## Features

- Interactive map of Chicago showing locations of social services
- Detailed information about each service including:
  - Operating hours
  - Available resources
  - Contact information
  - Real-time availability updates
- Filter services by category (food banks, shelters, healthcare, etc.)
- Search by location or service type
- Mobile-responsive design

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file in the root directory and add your Mapbox access token:
   ```
   VITE_MAPBOX_ACCESS_TOKEN=your_mapbox_token_here
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
